Job Description - .NET Developer

We are seeking a skilled .NET Developer to join our development team. As a .NET Developer, you will be responsible for designing, coding, and modifying software applications using the .NET framework. You will work closely with cross-functional teams to develop high-quality solutions that meet client requirements. If you have a strong background in .NET development and a passion for building innovative software applications, we would love to hear from you.

Responsibilities:

  • Design, code, and test software applications using .NET technologies
  • Collaborate with cross-functional teams to define project requirements
  • Participate in the full software development lifecycle, including analysis, design, development, testing, and deployment
  • Troubleshoot and debug issues in existing software applications
  • Perform code reviews and ensure code quality and best practices

Qualifications:

  • Bachelor's degree in Computer Science or a related field
  • Strong experience in .NET development
  • Proficient in C# and the .NET framework
  • Experience with ASP.NET, MVC, and Web API
  • Working knowledge of front-end web technologies such as HTML, CSS, and JavaScript
  • Experience with SQL databases and ORM frameworks
  • Familiarity with version control systems like Git
